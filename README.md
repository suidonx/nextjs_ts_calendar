## 概要
Next.jsとTypeScriptを使用して開発したカレンダーアプリです。  

https://nextjs-ts-calendar-6sdz0xp6h-suidonxs-projects.vercel.app

## 技術スタック
- Next.js 16.1.6
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- prettier
- date-fns
- shadcn/ui
- zod
- react-hook-form

## 機能一覧
- 週表示と月表示の切り替え
- 前週、翌週および前月、翌月への切り替え
- 日付ごとの予定作成、予定編集（タイトルのみ）、予定削除

## 細かい仕様
- ルートパスで今月のカレンダーを表示させる
- 今日の日付にマークをつける
